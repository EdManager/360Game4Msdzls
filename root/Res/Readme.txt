★★注意事项★★
1. V5版本的键鼠记忆模块需要系统屏幕缩放为100%才能正常录制（且导出文件不通用），但也可导入V6版本大厅录制的文件并正常使用。
2. 可通过再次安装的方式修改大厅及Flash版本，但若要修改安装路径，则需在安装位置找到 unins000.exe，卸载后再次安装，若卸载后仍有文件残留，手动删除整个 bin 文件夹即可。
3. Flash版本也可在目录 %USERPROFILE%\AppData\Roaming\360Game5\data 中手动替换：安装后会自动将所有flash版本文件均释放至安装目录\flash，将其中的 NPSWF34.dll 文件重命名为 NPSWF.dll 并替换data目录中的同名文件，极速模式Flash版本即改为34版。
4. 与上一条同理，兼容模式的Flash配置文件名为 Flash.ocx，可作类似替换，但V5版本需确保 AXsetup.exe（会在自定义安装V5且勾选启用兼容模式时安装，点确定即可）已安装才能成功使用、V6版本大概率不能成功替换（路径可能也会有所不同），且微端登录页可能无法使用兼容模式。
